Section 07Final Design & Decisions

The 25%. Make it count.

Every annotation connects to a product/business reason.
Never "looks better".
01Logo top-left, not center — matches LTR visual entry pattern for this demographic.
02Gold ≠ red. Red signals urgency; gold transfers existing LINE TV brand authority.
03Badge padding locked to 8pt grid; rhythm compounds across 14 components.
04Dark mode contrast audited at WCAG AAA — 40+ readers over-represented in evening sessions.
05Offline / loading fallback: skeleton badge, not removed badge — preserves visual anchor.
06Tap target: 44×44pt min. even when badge visually renders at 32×14.
Color tokens

Re-used 3 tokens from existing LINE TV system. Zero new primitives.

Type scale

Premium badge: 10.5pt · weight 600 · tracking +4%. Readable at 32pt min visual height.

Grid

8pt vertical, 4pt horizontal. Badge fits 4 grid cells wide in library module.

Motion

120ms cubic-bezier(0.2, 0, 0, 1) on badge enter. No exit animation — confidence, not decoration.

Light · default
Dark · 1:6.8 contrast
High-contrast · a11y

// story 01 · data team

"Won't removing 'Free' tank total clicks?"

"Are you sure moving away from the Free tag won't drop overall traffic?" — Data Lead

Response: 2-week staged A/B rollout with matched ad-spend baseline. Data — not opinion — closed the loop.

// story 02 · product manager

"We need 'free' for SEO."

"If we kill the Free tag, organic search discovery drops." — PM

Resolution: keep it in metadata, remove it from UI. Both goals protected. Compromise without softening the design.

// story 03 · operations team

"How do we upload now?"

"The ingest template changed. We need training." — Ops

Deliverable: Figma spec + 90-min workshop + macro'd template. Net: graphic design team load ↓ 18%.
